Ecuación cuadrática

Una ecuación de segundo grado o ecuación cuadrática de una variable es aquella que tiene la expresión general:

Ecuación de segundo grado
ax² + bx + c = 0, a ≠ 0

Donde x es la variable; mientras que a, b y c son valores constantes y corresponde con los coeficientes:

  • cuadrático (a)
  • lineal (b)
  • independiente (c)

Para una ecuación cuadrática con coeficientes reales o complejos existen siempre dos soluciones (x₁ y x₂), no necesariamente distintas, llamadas raíces, que pueden ser reales o complejas. Fórmula general para la obtención de raíces: x = (-b ± √(b² - 4ac)) ÷ (2a). Se utiliza el símbolo ± para indicar las dos soluciones.

C++

El primer programa que se suele crear al comenzar con un lenguaje de programación consiste en escribir algo en pantalla. Es muy frecuente que ese texto que escribe el programa sea un saludo al mundo que le rodea, un "Hola, mundo". En C++ sería así:

#include <iostream>

int main () {
    std::cout << "Hola, mundo";
    return 0;
}

Si no has programado antes, este programa debe sonar casi tan ilegible como si estuviese escrito en chino. Aun así, vamos a empezar a acercarnos al significado de cada frase sin entrar en mucho detalle:

  • #include deberá aparecer al principio de cualquier programa que escriba en pantalla o lea desde teclado
  • int main () indica que lo que viene a continuación es la parte principal, el cuerpo de nuestro programa
  • { } son las llaves, que indican el principio y el final de un bloque, en este caso del cuerpo de nuestro programa
  • std::cout << "Hola, mundo"; es
